Introduction

Consider the optimization problem \[\min_{x \in X} f(x),\] where \(f : X \to \bb{R}\).

The associated optimal value is defined as \[p^\star := \inf_{x \in X} f(x).\]

The infimum is used instead of the minimum since, in general, the function \(f\) may fail to attain its minimum over the set \(X\). That is, there may exist no point \(x^\star \in X\) such that \[f(x^\star) = \min_{x \in X} f(x).\]

Nevertheless, the set \(\{ f(x) : x \in X \} \subset \bb{R}\) always admits a greatest lower bound in \(\bb{R} \cup \{-\infty\}\), which is precisely the infimum.

If there exists a point \(x^\star \in X\) satisfying \[f(x^\star) = \inf_{x \in X} f(x),\] then the infimum is said to be attained, and we may equivalently write \[p^\star = \min_{x \in X} f(x).\] In this case, \(x^\star\) is called an optimal solution.

Consider the constrained optimization problem \[\begin{aligned} \min_{x \in \bb{R}^n} \quad & f(x) \\ \text{subject to} \quad & g(x) = 0, \end{aligned}\] where \(f : \bb{R}^n \to \bb{R}\) and \(g : \bb{R}^n \to \bb{R}^m\).

The associated feasible set is defined by \[X := \{ x \in \bb{R}^n : g(x) = 0 \}.\]

A point \(x^\star \in X\) is called a local minimum of the constrained optimization problem if there exists \(\delta > 0\) such that \[f(x^\star) \le f(x) \quad \text{for all } x \in X \cap B(x^\star,\delta),\] where \[B(x^\star,\delta) := \{ x \in \bb{R}^n : \|x - x^\star\| < \delta \}\] denotes the open ball of radius \(\delta\) centered at \(x^\star\).

Equivalently, \(x^\star\) is a local minimum if no feasible point sufficiently close to \(x^\star\) achieves a strictly smaller objective value.

Consider the optimization problem \[\min_{x \in \bb{R}} \left\{ \frac{1}{2}(x-a)^2 + \tau |x| \right\}, \qquad \tau > 0.\]

The objective function is convex but non-smooth due to the absolute value term. The subdifferential of \(|x|\) is the set-valued mapping \[\partial |x| = \begin{cases} \{1\}, & x > 0, \\[6pt] [-1,\,1], & x = 0, \\[6pt] \{-1\}, & x < 0. \end{cases}\]

Define \[\phi(x) := \frac{1}{2}(x-a)^2 + \tau |x|.\] Since \(\phi\) is convex, a point \(x^\star \in \bb{R}\) is optimal if and only if \[0 \in \partial \phi(x^\star) = (x^\star - a) + \tau \partial |x^\star|.\] Equivalently, \[a - x^\star \in \tau \partial |x^\star|.\]

A case-by-case analysis yields \[x^\star = \begin{cases} a - \tau, & a > \tau, \\[6pt] 0, & |a| \le \tau, \\[6pt] a + \tau, & a < -\tau. \end{cases}\]

This solution can be written compactly as the soft-thresholding operator \[x^\star = \mathrm{sign}(a)\,\max\{ |a| - \tau,\, 0 \}.\]

Inclusion Problems, Convexity, and Smoothness

Let \(X\) be a real Euclidean space. An inclusion problem consists in finding \[x \in X \quad \text{such that} \quad 0 \in T(x),\] where \(T : X \rightrightarrows X\) is a set-valued operator.

Many convex optimization problems can be written as inclusion problems by taking \(T = \partial f\), where \(\partial f\) denotes the subdifferential of a convex function.

Let \(C \subset X\) be a convex set and let \(f : X \to \bb{R}\) be a convex function. The problem \[\min_{x \in C} f(x)\] is called a convex optimization problem.

If both the objective function \(f\) and the constraint set \(C\) are convex, then every local minimizer is a global minimizer.

Let \(f : \bb{R}^n \to \bb{R}\) be twice continuously differentiable. If \[\nabla^2 f(x) \succeq 0 \quad \text{for all } x \in \bb{R}^n,\] then \(f\) is convex. If \[\nabla^2 f(x) \succ 0 \quad \text{for all } x \in \bb{R}^n,\] then \(f\) is strictly convex.

Euclidean spaces and induced norms

A real Euclidean space \(X\) is a real vector space equipped with an inner product \[\iprod{x}{y}.\] The induced norm is defined by \[\|x\| := \sqrt{\iprod{x}{x}}.\]

The \(\ell^2\)-norm on \(\bb{R}^n\) is induced by the standard inner product. The \(\ell^1\)-norm is not induced by any inner product.

The norm induces a metric \[d(x,y) := \|x-y\|,\] which defines a topology on \(X\).

A sequence \((x_k) \subset X\) converges to \(x \in X\) if \[\|x_k - x\| \to 0.\]

A function \(f : X \to \bb{R}\) is continuous at \(x \in X\) if \[x_k \to x \quad \Rightarrow \quad f(x_k) \to f(x).\]

Let \(X\) be a set and let \(\tau\) be a collection of subsets of \(X\). The pair \((X,\tau)\) is called a topological space if

  • \(\varnothing \in \tau\) and \(X \in \tau\),

  • the union of any collection of sets in \(\tau\) belongs to \(\tau\),

  • the intersection of any finite collection of sets in \(\tau\) belongs to \(\tau\).

The elements of \(\tau\) are called open sets.

Let \((X,\tau)\) be a topological space and let \((x_k) \subset X\). We say that \(x_k\) converges to \(x \in X\) if for every open set \(U \in \tau\) such that \(x \in U\), there exists \(N \in \bb{N}\) for which \[x_k \in U \quad \text{for all } k \ge N.\]

This definition of convergence depends only on the topology \(\tau\) and does not require any notion of distance or norm. In particular, convergence can be defined in spaces that are not metric spaces.

Every convex function \(f : \bb{R}^n \to \bb{R}\) is continuous.

For \(0 < p < 1\), the set \[B_p := \{ x \in \bb{R}^n : \|x\|_p \le 1 \}\] is not convex.

A function \(f : \bb{R}^n \to \bb{R}\) is said to be differentiable at \(x \in \bb{R}^n\) if there exists \(z \in \bb{R}^n\) such that \[f(x+h) = f(x) + \iprod{z}{h} + o(\|h\|) \quad \text{as } h \to 0.\] In this case, \(z = \nabla f(x)\). where The term \(o(\|h\|)\) satisfies \[\lim_{\|h\|\to 0} \frac{o(\|h\|)}{\|h\|} = 0.\]

Let \(f : \bb{R}^n \to \bb{R}\) be differentiable at \(x \in \bb{R}^n\) in the sense that there exists \(z \in \bb{R}^n\) such that \[f(x+h) = f(x) + \iprod{z}{h} + o(\|h\|) \quad \text{as } h \to 0,\] where \[\lim_{\|h\|\to 0} \frac{o(\|h\|)}{\|h\|} = 0.\]

  1. Show that for a fixed inner product \(\iprod{\cdot}{\cdot}\), the vector \(z\) is unique.

  2. Show that the vector \(z\) depends on the choice of inner product.

  3. Show that the notion of differentiability does not depend on the choice of inner product on \(\bb{R}^n\).

  4. Show that all norms on \(\bb{R}^n\) are equivalent.

  5. Prove the Riesz representation theorem: every linear functional on a Euclidean space can be represented as an inner product.

Solution.

  1. Uniqueness: Assume that there exist \(z_1, z_2 \in \bb{R}^n\) such that \[f(x+h) = f(x) + \iprod{z_1}{h} + o(\|h\|)\] and \[f(x+h) = f(x) + \iprod{z_2}{h} + o(\|h\|).\] Subtracting the two expressions gives \[\iprod{z_1 - z_2}{h} = o(\|h\|).\] Dividing by \(\|h\|\) and letting \(h \to 0\), \[\frac{\iprod{z_1 - z_2}{h}}{\|h\|} \to 0.\] Choosing \(h = z_1 - z_2\) yields \[\|z_1 - z_2\|^2 = 0,\] hence \(z_1 = z_2\).

  2. Dependence on inner product: Let \(f(x) = a^\TT x\), where \(a \in \bb{R}^n\). For the standard inner product, \[f(x+h) = f(x) + \iprod{a}{h},\] so \(z = a\).

    Now consider a different inner product \[\iprod{x}{y}_M := \iprod{Mx}{y},\] where \(M\) is symmetric positive definite. Then \[f(x+h) = f(x) + \iprod{M^{-1}a}{h}_M.\] Hence the representing vector \(z\) changes with the inner product.

  3. Independence of differentiability: Differentiability is equivalent to the existence of a linear map \(L : \bb{R}^n \to \bb{R}\) such that \[f(x+h) = f(x) + L(h) + o(\|h\|).\] This definition does not involve any inner product.

    By the Riesz representation theorem, for any chosen inner product, there exists a unique vector \(z\) such that \[L(h) = \iprod{z}{h}.\] Thus, while the vector \(z\) depends on the inner product, the existence of the derivative does not.

  4. Norm equivalence: Let \(\|\cdot\|_a\) and \(\|\cdot\|_b\) be two norms on \(\bb{R}^n\). To show they are equivalent, we need to find \(c, C > 0\) such that \(c \|x\|_a \le \|x\|_b \le C \|x\|_a\) for all \(x\).

    The unit sphere \(S = \{x : \|x\|_a = 1\}\) is compact. The function \(x \mapsto \|x\|_b\) is continuous on \(S\), so it attains a minimum \(c > 0\) and a maximum \(C > 0\). By homogeneity of norms, the result follows for all \(x\).

  5. Riesz representation theorem: Let \(\{e_1,\dots,e_n\}\) be an orthonormal basis of \(X\). Define \[z := \sum_{i=1}^n L(e_i)\, e_i.\] Then for any \(h = \sum_i h_i e_i\), \[\iprod{z}{h} = \sum_{i=1}^n L(e_i) h_i = L(h).\] Uniqueness follows from non-degeneracy of the inner product.

Let \(X \subset \bb{R}^n\) be a convex set and let \(f : X \to \bb{R}\) be convex and differentiable. If \(x^\star \in X\) satisfies \[\nabla f(x^\star) = 0,\] then \(x^\star\) is a global minimizer of \(f\) on \(X\).

Proof. Since \(f\) is convex, for any \(x, y \in X\) and any \(t \in [0,1]\), \[f(tx + (1-t)y) \le t f(x) + (1-t) f(y).\]

Fix \(x \in X\) and define the function \[\varphi(t) := f(x^\star + t(x - x^\star)), \qquad t \in [0,1].\] By convexity of \(f\), the function \(\varphi\) is convex on \([0,1]\).

Since \(f\) is differentiable, \(\varphi\) is differentiable and \[\varphi'(0) = \iprod{\nabla f(x^\star)}{x - x^\star}.\]

By assumption, \(\nabla f(x^\star) = 0\), hence \[\varphi'(0) = 0.\]

For a convex function on an interval, the derivative is monotone non-decreasing. Therefore, for all \(t \in [0,1]\), \[\varphi(t) \ge \varphi(0).\]

In particular, at \(t = 1\), \[f(x) = \varphi(1) \ge \varphi(0) = f(x^\star).\]

Since \(x \in X\) was arbitrary, \(x^\star\) is a global minimizer of \(f\). ◻

Unconstrained and Constrained Optimization

An unconstrained optimization problem is of the form \[\min_{x \in X} f(x),\] where typically \(X = \bb{R}^n\) and \(f : \bb{R}^n \to \bb{R}\).

In this case, every point in \(X\) is feasible, and optimality is characterized by stationarity conditions such as \[\nabla f(x^\star) = 0.\]

A constrained optimization problem is given by \[\min_{x \in C} f(x),\] where \(C \subset \bb{R}^n\) is a constraint set and \(f : \bb{R}^n \to \bb{R}\).

The set \[C := \{ x \in \bb{R}^n : \text{all constraints are satisfied} \}\] is called the feasible set. Points in \(C\) are called feasible points.

Let \(C \subset \bb{R}^n\). The indicator function \(\delta_C : \bb{R}^n \to \bb{R} \cup \{+\infty\}\) is defined as \[\delta_C(x) := \begin{cases} 0, & x \in C, \\[4pt] +\infty, & x \notin C. \end{cases}\]

Then the constrained problem \[\min_{x \in C} f(x)\] is equivalent to the unconstrained problem \[\min_{x \in \bb{R}^n} \bigl( f(x) + \delta_C(x) \bigr).\]

Consider the constrained problem \[\begin{aligned} \min_{x \in \bb{R}^n} \quad & f(x) \\ \text{subject to} \quad & g(x) \le 0, \end{aligned}\] where \(g : \bb{R}^n \to \bb{R}^m\). The Lagrangian associated with the problem is \[\mathcal{L}(x,\lambda) := f(x) + \iprod{\lambda}{g(x)}, \qquad \lambda \in \bb{R}^m_+.\]

If \(x^\star\) is a solution of the constrained problem and \(\lambda^\star \ge 0\) is a corresponding Lagrange multiplier, then \[\mathcal{L}(x^\star,\lambda) \le \mathcal{L}(x^\star,\lambda^\star) \le \mathcal{L}(x,\lambda^\star)\] for all feasible \(x\) and all \(\lambda \ge 0\).

Thus, optimal solutions correspond to saddle points of the Lagrangian.

Interior-point conditions (such as Slater’s condition) ensure that Lagrange multipliers exist and that optimality conditions are valid.

Consider the problem \[\min_{x \in \bb{R}} x \quad \text{subject to} \quad x \ge 0.\] The optimal solution is \(x^\star = 0\), which lies on the boundary of the feasible set.

If we perturb the constraint slightly, the gradient of the objective cannot be balanced by any Lagrange multiplier unless the feasible set has nonempty interior. This failure prevents the derivation of meaningful optimality conditions.

Interior feasibility guarantees that constraints are not degenerate and ensures the validity of duality and KKT conditions. Without interior points, multipliers may fail to exist or be unbounded.

Let \(\bb{X}\) be a normed vector space.

  • \(\Gamma(\bb{X}) := \{ f : \bb{X} \to \bb{R} \mid f \text{ is convex} \}\).

  • \(C^0(\bb{X})\) denotes the space of continuous functions on \(\bb{X}\).

  • \(C^1(\bb{X})\) denotes the space of continuously differentiable functions on \(\bb{X}\).

  • A function \(f : \bb{X} \to \bb{R}\) is called \(\beta\)-Lipschitz if \[|f(\bd{x}) - f(\bd{y})| \le \beta \|\bd{x} - \bd{y}\| \quad \text{for all } \bd{x},\bd{y} \in \bb{X}.\]

  • A function \(f : \bb{X} \to \bb{R}\) is called \(\beta\)-smooth if \(f \in C^1(\bb{X})\) and \[\|\nabla f(\bd{x}) - \nabla f(\bd{y})\| \le \beta \|\bd{x} - \bd{y}\| \quad \text{for all } \bd{x},\bd{y} \in \bb{X}.\]

  • \[C^1_\beta(\bb{X}) := \{ f : \bb{X} \to \bb{R} \mid f \in C^1(\bb{X}) \text{ and } \nabla f \text{ is } \beta\text{-Lipschitz} \}.\]

  • Let \(\bb{X}=\bb{R}^2\). The function \[f(\bd{x}) = \tfrac12 \|\bd{x}\|_2^2\] is \(1\)-smooth.

  • Let \(Q \in \bb{R}^{n \times n}\) be symmetric positive semidefinite. The quadratic function \[f(\bd{x}) = \tfrac12 \bd{x}^\top Q \bd{x}\] is \(\lambda_{\max}(Q)\)-smooth.

  • The function \[f(\bd{x}) = \|\bd{x}\|_2^2\] is \(4\)-smooth on the unit ball \[\{ \bd{x} \in \bb{R}^n : \|\bd{x}\|_2 \le 1 \}.\]

Let \(f : \bb{R}^n \to \bb{R}\) be continuously differentiable. Then \(f\) is \(\beta\)-Lipschitz (with respect to the Euclidean norm) if and only if \[\|\nabla f(\bd{x})\|_2 \le \beta \quad \text{for all } \bd{x} \in \bb{R}^n.\]

Proof. (\(\Rightarrow\)) Assume \(f\) is \(\beta\)-Lipschitz. Fix \(\bd{x} \in \bb{R}^n\) and let \(\bd{v} \in \bb{R}^n\) be any unit vector. For \(t \in \bb{R}\), \[|f(\bd{x} + t\bd{v}) - f(\bd{x})| \le \beta |t|.\] Dividing by \(|t|\) and letting \(t \to 0\), we obtain \[|\nabla f(\bd{x})^\top \bd{v}| \le \beta.\] Taking the supremum over all unit vectors \(\bd{v}\) yields \[\|\nabla f(\bd{x})\|_2 \le \beta.\]

(\(\Leftarrow\)) Assume \(\|\nabla f(\bd{x})\|_2 \le \beta\) for all \(\bd{x} \in \bb{R}^n\). For any \(\bd{x}, \bd{y} \in \bb{R}^n\), define \[\bld{\gamma}(t) = \bd{x} + t(\bd{y} - \bd{x}), \quad t \in [0,1].\] By the fundamental theorem of calculus, \[f(\bd{y}) - f(\bd{x}) = \int_0^1 \nabla f(\bld{\gamma}(t))^\top (\bd{y} - \bd{x}) \, dt.\] Applying the Cauchy–Schwarz inequality, \[|f(\bd{y}) - f(\bd{x})| \le \int_0^1 \|\nabla f(\bld{\gamma}(t))\|_2 \, \|\bd{y} - \bd{x}\|_2 \, dt \le \beta \|\bd{y} - \bd{x}\|_2.\] Thus, \(f\) is \(\beta\)-Lipschitz. ◻

Let \(f : \bb{R}^n \to \bb{R}\) be differentiable. The following statements are equivalent:

  1. \(f\) is convex.

  2. For all \(\bd{x}, \bd{y} \in \bb{R}^n\), \[f(\bd{y}) \ge f(\bd{x}) + \langle \nabla f(\bd{x}), \bd{y} - \bd{x} \rangle.\]

  3. For all \(\bd{x}, \bd{y} \in \bb{R}^n\), \[\langle \nabla f(\bd{x}) - \nabla f(\bd{y}), \bd{x} - \bd{y} \rangle \ge 0.\]

Proof. We prove the equivalence by showing \((1)\Rightarrow(2)\Rightarrow(3)\Rightarrow(1)\).

\((1)\Rightarrow(2)\). Assume that \(f\) is convex. Fix \(\bd{x}, \bd{y} \in \bb{R}^n\) and define \[\phi(t) := f(\bd{x} + t(\bd{y} - \bd{x})), \quad t \in [0,1].\] Since \(f\) is convex and differentiable, \(\phi\) is convex and differentiable. By convexity of \(\phi\), \[\phi(1) \ge \phi(0) + \phi'(0).\] Using the chain rule, \[\phi'(0) = \langle \nabla f(\bd{x}), \bd{y} - \bd{x} \rangle.\] Substituting \(\phi(1)=f(\bd{y})\) and \(\phi(0)=f(\bd{x})\) yields \[f(\bd{y}) \ge f(\bd{x}) + \langle \nabla f(\bd{x}), \bd{y} - \bd{x} \rangle.\]

\((2)\Rightarrow(3)\). Assume that (2) holds. Applying (2) with \((\bd{x},\bd{y})\) gives \[f(\bd{y}) \ge f(\bd{x}) + \langle \nabla f(\bd{x}), \bd{y} - \bd{x} \rangle.\] Applying (2) with \((\bd{y},\bd{x})\) gives \[f(\bd{x}) \ge f(\bd{y}) + \langle \nabla f(\bd{y}), \bd{x} - \bd{y} \rangle.\] Adding the two inequalities and simplifying yields \[\langle \nabla f(\bd{x}) - \nabla f(\bd{y}), \bd{x} - \bd{y} \rangle \ge 0.\]

\((3)\Rightarrow(1)\). Assume that (3) holds. Fix \(\bd{x}, \bd{y} \in \bb{R}^n\) and define \[\bld{\gamma}(t) := (1-t)\bd{x} + t\bd{y}, \quad t \in [0,1],\] and \[\phi(t) := f(\bld{\gamma}(t)).\] By the chain rule, \[\phi'(t) = \langle \nabla f(\bld{\gamma}(t)), \bd{y} - \bd{x} \rangle.\] Applying (3) with \(\bd{x}=\bld{\gamma}(t)\) and \(\bd{y}=\bd{x}\) gives \[\langle \nabla f(\bld{\gamma}(t)) - \nabla f(\bd{x}), \bld{\gamma}(t) - \bd{x} \rangle \ge 0.\] Since \(\bld{\gamma}(t)-\bd{x}=t(\bd{y}-\bd{x})\), this implies \[\phi'(t) \ge \phi'(0).\] Thus \(\phi'\) is nondecreasing on \([0,1]\), and hence \(\phi\) is convex. Therefore, \[f(\bld{\gamma}(t)) \le (1-t)f(\bd{x}) + t f(\bd{y}), \quad \text{for all } t \in [0,1].\] This is precisely the definition of convexity of \(f\). ◻

When \(n=1\), condition (3) reduces to \[(f'(x) - f'(y))(x - y) \ge 0,\] which is equivalent to monotonicity of \(f'\). Hence, a differentiable function \(f : \bb{R} \to \bb{R}\) is convex if and only if its derivative is nondecreasing.

Let \(f \in C^1_\beta(\bb{X})\), where \(\bb{X} \subset \bb{R}^n\). Then, for all \(\bd{x}, \bd{y} \in \bb{X}\), \[f(\bd{y}) \le f(\bd{x}) + \langle \nabla f(\bd{x}), \bd{y} - \bd{x} \rangle + \frac{\beta}{2}\|\bd{y} - \bd{x}\|_2^2.\]

Proof. Fix \(\bd{x}, \bd{y} \in \bb{X}\) and define \[\bld{\gamma}(t) := \bd{x} + t(\bd{y} - \bd{x}), \quad t \in [0,1],\] and \[\phi(t) := f(\bld{\gamma}(t)).\] By the chain rule, \[\phi'(t) = \langle \nabla f(\bld{\gamma}(t)), \bd{y} - \bd{x} \rangle.\] Using the fundamental theorem of calculus, \[f(\bd{y}) - f(\bd{x}) = \phi(1) - \phi(0) = \int_0^1 \phi'(t)\, dt.\] Add and subtract \(\nabla f(\bd{x})\) inside the integrand: \[\begin{aligned} f(\bd{y}) - f(\bd{x}) & = \int_0^1 \langle \nabla f(\bd{x}), \bd{y} - \bd{x} \rangle \, dt \\ & \quad + \int_0^1 \langle \nabla f(\bld{\gamma}(t)) - \nabla f(\bd{x}), \bd{y} - \bd{x} \rangle \, dt. \end{aligned}\] The first term gives \[\langle \nabla f(\bd{x}), \bd{y} - \bd{x} \rangle.\] For the second term, by Cauchy–Schwarz and \(\beta\)-Lipschitz continuity of \(\nabla f\), \[\begin{aligned} \left| \langle \nabla f(\bld{\gamma}(t)) - \nabla f(\bd{x}), \bd{y} - \bd{x} \rangle \right| & \le \|\nabla f(\bld{\gamma}(t)) - \nabla f(\bd{x})\|_2 \|\bd{y} - \bd{x}\|_2 \\ & \le \beta t \|\bd{y} - \bd{x}\|_2^2. \end{aligned}\] Integrating over \(t \in [0,1]\) yields \[\int_0^1 \beta t \|\bd{y} - \bd{x}\|_2^2 \, dt = \frac{\beta}{2}\|\bd{y} - \bd{x}\|_2^2.\] Combining all terms gives \[f(\bd{y}) \le f(\bd{x}) + \langle \nabla f(\bd{x}), \bd{y} - \bd{x} \rangle + \frac{\beta}{2}\|\bd{y} - \bd{x}\|_2^2.\] ◻

Let \(f \in C^1_\beta(\bb{X})\) and let \(\gamma \in (0, 2/\beta]\). Consider the gradient descent iteration \[\bd{x}_{k+1} = \bd{x}_k - \gamma \nabla f(\bd{x}_k), \quad \bd{x}_0 \in \bb{X}.\] Then, for all \(k \ge 0\), \[f(\bd{x}_{k+1}) \le f(\bd{x}_k),\] and more precisely, \[f(\bd{x}_{k+1}) \le f(\bd{x}_k) - \gamma\Bigl(1 - \tfrac{\beta\gamma}{2}\Bigr) \|\nabla f(\bd{x}_k)\|_2^2.\]

Proof. Fix \(k \ge 0\). Apply Lemma [lem:smoothness] (the smoothness upper bound) with \[\bd{x} = \bd{x}_k, \qquad \bd{y} = \bd{x}_{k+1}.\] Since \[\bd{x}_{k+1} - \bd{x}_k = - \gamma \nabla f(\bd{x}_k),\] Lemma [lem:smoothness] yields \[\begin{aligned} f(\bd{x}_{k+1}) & \le f(\bd{x}_k) + \langle \nabla f(\bd{x}_k), \bd{x}_{k+1} - \bd{x}_k \rangle + \frac{\beta}{2}\|\bd{x}_{k+1} - \bd{x}_k\|_2^2 \\ & = f(\bd{x}_k) - \gamma \|\nabla f(\bd{x}_k)\|_2^2 + \frac{\beta}{2}\gamma^2 \|\nabla f(\bd{x}_k)\|_2^2. \end{aligned}\] Rearranging gives \[f(\bd{x}_{k+1}) \le f(\bd{x}_k) - \gamma\Bigl(1 - \tfrac{\beta\gamma}{2}\Bigr) \|\nabla f(\bd{x}_k)\|_2^2.\] If \(\gamma \in (0,2/\beta]\), then \(1 - \tfrac{\beta\gamma}{2} \ge 0\), and hence \(f(\bd{x}_{k+1}) \le f(\bd{x}_k)\). ◻

Let \(f \in C^1_\beta(\bb{R}^n)\) be convex and let \(\bd{x}^\ast \in \arg\min f\) with \(p^\ast := f(\bd{x}^\ast)\). Consider gradient descent with step size \(\gamma \in (0, 1/\beta]\): \[\bd{x}_{k+1} = \bd{x}_k - \gamma \nabla f(\bd{x}_k).\] Define \[\delta_k := f(\bd{x}_k) - p^\ast.\] Then:

  1. \(\{f(\bd{x}_k)\}\) is nonincreasing and bounded below by \(p^\ast\);

  2. \(\sum_{k=0}^\infty \|\nabla f(\bd{x}_k)\|_2^2 < \infty\);

  3. there exists \(C > 0\) such that \[\delta_k \le \frac{C}{k} \quad \text{for all } k \ge 1.\]

Proof. By the descent lemma, for \(\gamma \le 1/\beta\), \[f(\bd{x}_{k+1}) \le f(\bd{x}_k) - \frac{\gamma}{2}\|\nabla f(\bd{x}_k)\|_2^2.\] Subtracting \(p^\ast\) from both sides yields \[\delta_{k+1} \le \delta_k - \frac{\gamma}{2}\|\nabla f(\bd{x}_k)\|_2^2. \tag{1}\] Hence \(\{\delta_k\}\) is nonincreasing and bounded below by \(0\), and therefore \[\delta_k \to v \ge 0.\]

Summing inequality (1) from \(k=0\) to \(N-1\) gives \[\delta_N \le \delta_0 - \frac{\gamma}{2} \sum_{k=0}^{N-1} \|\nabla f(\bd{x}_k)\|_2^2.\] Since \(\delta_N \ge 0\), we obtain \[\sum_{k=0}^{\infty} \|\nabla f(\bd{x}_k)\|_2^2 \le \frac{2}{\gamma}\delta_0 < \infty. \tag{2}\]

By convexity of \(f\), \[f(\bd{x}_k) - p^\ast \le \langle \nabla f(\bd{x}_k), \bd{x}_k - \bd{x}^\ast \rangle.\] Using the Cauchy–Schwarz inequality, \[\delta_k \le \|\nabla f(\bd{x}_k)\|_2 \|\bd{x}_k - \bd{x}^\ast\|_2.\] Since \(\{f(\bd{x}_k)\}\) is decreasing and \(f\) is coercive on level sets, there exists \(R > 0\) such that \[\|\bd{x}_k - \bd{x}^\ast\|_2 \le R \quad \text{for all } k.\]

The boundedness of \(\{\bd{x}_k\}\) is established later using Fejér monotonicity; see Remark [rem:fejer-boundedness].

Thus, \[\delta_k \le R \|\nabla f(\bd{x}_k)\|_2. \tag{3}\]

Combining (1) and (3), we obtain \[\delta_{k+1} \le \delta_k - \frac{\gamma}{2R^2} \delta_k^2.\] This implies \[\frac{1}{\delta_{k+1}} \ge \frac{1}{\delta_k} + \frac{\gamma}{2R^2}.\] Iterating, \[\frac{1}{\delta_k} \ge \frac{1}{\delta_0} + k\frac{\gamma}{2R^2}.\] Taking reciprocals yields \[\delta_k \le \frac{2R^2}{\gamma k} = \frac{C}{k}, \quad C := \frac{2R^2}{\gamma}.\] ◻

The boundedness assumption used in the proof above is not restrictive. In fact, for \(\gamma \in (0,1/\beta]\), the gradient descent iterates \(\{\bd{x}_k\}\) are Fejér monotone with respect to \(\arg\min f\), which implies boundedness and convergence.

Let \(f : \bb{R}^n \to \bb{R}\) be convex and continuously differentiable. If \(\nabla f\) is \(\beta\)-Lipschitz, then \(\nabla f\) is \(\frac{1}{\beta}\)-cocoercive, i.e., for all \(\bd{x}, \bd{y} \in \bb{R}^n\), \[\langle \nabla f(\bd{x}) - \nabla f(\bd{y}), \bd{x} - \bd{y} \rangle \ge \frac{1}{\beta} \|\nabla f(\bd{x}) - \nabla f(\bd{y})\|_2^2.\]

Proof. Since \(f \in C^1_\beta(\bb{R}^n)\), the descent lemma gives, for all \(\bd{x}, \bd{y} \in \bb{R}^n\), \[f(\bd{y}) \le f(\bd{x}) + \langle \nabla f(\bd{x}), \bd{y} - \bd{x} \rangle + \frac{\beta}{2}\|\bd{y} - \bd{x}\|_2^2. \tag{1}\] Exchanging the roles of \(\bd{x}\) and \(\bd{y}\) yields \[f(\bd{x}) \le f(\bd{y}) + \langle \nabla f(\bd{y}), \bd{x} - \bd{y} \rangle + \frac{\beta}{2}\|\bd{x} - \bd{y}\|_2^2. \tag{2}\] Adding (1) and (2) gives \[\langle \nabla f(\bd{x}) - \nabla f(\bd{y}), \bd{x} - \bd{y} \rangle \le \beta \|\bd{x} - \bd{y}\|_2^2. \tag{3}\]

On the other hand, since \(\nabla f\) is \(\beta\)-Lipschitz, \[\|\nabla f(\bd{x}) - \nabla f(\bd{y})\|_2^2 \le \beta \langle \nabla f(\bd{x}) - \nabla f(\bd{y}), \bd{x} - \bd{y} \rangle.\] Rearranging yields the desired inequality: \[\langle \nabla f(\bd{x}) - \nabla f(\bd{y}), \bd{x} - \bd{y} \rangle \ge \frac{1}{\beta} \|\nabla f(\bd{x}) - \nabla f(\bd{y})\|_2^2.\] ◻

Let \(\mathcal{C} \subset \bb{R}^n\) be nonempty. A sequence \(\{\bd{x}_k\} \subset \bb{R}^n\) is said to be Fejér monotone with respect to \(\mathcal{C}\) if \[\|\bd{x}_{k+1} - \bd{z}\|_2 \le \|\bd{x}_k - \bd{z}\|_2 \quad \text{for all } \bd{z} \in \mathcal{C} \text{ and all } k \ge 0.\]

Let \(f \in C^1_\beta(\bb{R}^n)\) be convex and let \(\mathcal{X}^\ast := \arg\min f \neq \varnothing\). Assume \(\gamma \in (0,1/\beta]\) and consider \[\bd{x}_{k+1} = \bd{x}_k - \gamma \nabla f(\bd{x}_k).\] Then \(\{\bd{x}_k\}\) is Fejér monotone with respect to \(\mathcal{X}^\ast\).

Proof. Fix \(\bd{x}^\ast \in \mathcal{X}^\ast\). We compute \[\bd{x}_{k+1}-\bd{x}^\ast = \bd{x}_k-\bd{x}^\ast - \gamma \nabla f(\bd{x}_k).\] Taking squared norms, \[\begin{aligned} \|\bd{x}_{k+1}-\bd{x}^\ast\|_2^2 & = \|\bd{x}_k-\bd{x}^\ast\|_2^2 -2\gamma\langle\nabla f(\bd{x}_k),\bd{x}_k-\bd{x}^\ast\rangle +\gamma^2\|\nabla f(\bd{x}_k)\|_2^2. \end{aligned}\]

By convexity of \(f\), \[\langle\nabla f(\bd{x}_k),\bd{x}_k-\bd{x}^\ast\rangle \ge f(\bd{x}_k)-f(\bd{x}^\ast). \tag{1}\]

Since \(f\) is convex and \(\beta\)-smooth, the Baillon–Haddad  [lem:baillon-haddad] implies that \(\nabla f\) is \(1/\beta\)-cocoercive. In particular, using \(\nabla f(\bd{x}^\ast)=0\), we obtain \[\|\nabla f(\bd{x}_k)\|_2^2 \le \beta\, \langle\nabla f(\bd{x}_k),\bd{x}_k-\bd{x}^\ast\rangle. \tag{2}\]

Substituting (1) and (2) into the norm identity yields \[\begin{aligned} \|\bd{x}_{k+1}-\bd{x}^\ast\|_2^2 & \le \|\bd{x}_k-\bd{x}^\ast\|_2^2 -2\gamma\langle\nabla f(\bd{x}_k),\bd{x}_k-\bd{x}^\ast\rangle +\beta\gamma^2\langle\nabla f(\bd{x}_k),\bd{x}_k-\bd{x}^\ast\rangle \\ & = \|\bd{x}_k-\bd{x}^\ast\|_2^2 -\gamma(2-\beta\gamma) \langle\nabla f(\bd{x}_k),\bd{x}_k-\bd{x}^\ast\rangle. \end{aligned}\]

Since \(\gamma \le 1/\beta\), the coefficient \((2-\beta\gamma)\) is nonnegative, and using (1) once more, we conclude \[\|\bd{x}_{k+1}-\bd{x}^\ast\|_2^2 \le \|\bd{x}_k-\bd{x}^\ast\|_2^2.\] Hence, \[\|\bd{x}_{k+1}-\bd{x}^\ast\|_2 \le \|\bd{x}_k-\bd{x}^\ast\|_2,\] which proves Fejér monotonicity. ◻

Let \(\{\bd{x}_k\}\) be Fejér monotone with respect to a closed set \(\mathcal{C}\subset\bb{R}^n\). If there exists a subsequence \(\bd{x}_{k_j}\to\bd{x}^\ast\in\mathcal{C}\), then \(\bd{x}_k\to\bd{x}^\ast\).

Proof. Since \(\{\bd{x}_k\}\) is Fejér monotone with respect to \(\mathcal{C}\), it is bounded. Hence, by the Bolzano-Weierstrass theorem, it admits a convergent subsequence \(\bd{x}_{k_j}\to\bd{x}^\ast\in\mathcal{C}\).

Fix \(\bd{z}=\bd{x}^\ast\). By Fejér monotonicity, \[\|\bd{x}_{k+1}-\bd{x}^\ast\|_2 \le \|\bd{x}_k-\bd{x}^\ast\|_2,\] so the sequence \(\|\bd{x}_k-\bd{x}^\ast\|_2\) is nonincreasing and converges to some \(\ell\ge0\).

Along the subsequence, \[\|\bd{x}_{k_j}-\bd{x}^\ast\|_2 \to 0,\] hence \(\ell=0\). Therefore, \[\bd{x}_k \to \bd{x}^\ast.\] ◻

Let \(f \in C^1_\beta(\bb{R}^n)\) be convex and let \(\bd{x}^\ast \in \arg\min f\). Assume \(\gamma \in (0, 1/\beta]\) and consider the gradient descent iteration \[\bd{x}_{k+1} = \bd{x}_k - \gamma \nabla f(\bd{x}_k).\] Then the sequence \(\{\bd{x}_k\}\) is Fejér monotone with respect to the solution set \(\mathcal{X}^\ast := \arg\min f\), i.e., \[\|\bd{x}_{k+1} - \bd{x}^\ast\|_2 \le \|\bd{x}_k - \bd{x}^\ast\|_2 \quad \text{for all } \bd{x}^\ast \in \mathcal{X}^\ast,\] and in particular \(\{\bd{x}_k\}\) is bounded.

Proof. Fix \(\bd{x}^\ast \in \mathcal{X}^\ast\). Using the update rule, \[\bd{x}_{k+1} - \bd{x}^\ast = \bd{x}_k - \bd{x}^\ast - \gamma \nabla f(\bd{x}_k).\] Taking squared norms, \[\begin{aligned} \|\bd{x}_{k+1} - \bd{x}^\ast\|_2^2 & = \|\bd{x}_k - \bd{x}^\ast\|_2^2 - 2\gamma \langle \nabla f(\bd{x}_k), \bd{x}_k - \bd{x}^\ast \rangle + \gamma^2 \|\nabla f(\bd{x}_k)\|_2^2. \end{aligned}\] By convexity of \(f\) and optimality of \(\bd{x}^\ast\), \[\langle \nabla f(\bd{x}_k), \bd{x}_k - \bd{x}^\ast \rangle \ge f(\bd{x}_k) - f(\bd{x}^\ast). \tag{1}\] By \(\beta\)-smoothness, \[\|\nabla f(\bd{x}_k)\|_2^2 \le 2\beta \bigl(f(\bd{x}_k) - f(\bd{x}^\ast)\bigr). \tag{2}\] Substituting (1) and (2) into the squared norm expression yields \[\begin{aligned} \|\bd{x}_{k+1} - \bd{x}^\ast\|_2^2 & \le \|\bd{x}_k - \bd{x}^\ast\|_2^2 - 2\gamma \bigl(f(\bd{x}_k) - f(\bd{x}^\ast)\bigr) + 2\beta\gamma^2 \bigl(f(\bd{x}_k) - f(\bd{x}^\ast)\bigr) \\ & = \|\bd{x}_k - \bd{x}^\ast\|_2^2 - 2\gamma(1 - \beta\gamma) \bigl(f(\bd{x}_k) - f(\bd{x}^\ast)\bigr). \end{aligned}\] Since \(\gamma \le 1/\beta\), the coefficient \(2\gamma(1-\beta\gamma)\) is nonnegative, and hence \[\|\bd{x}_{k+1} - \bd{x}^\ast\|_2^2 \le \|\bd{x}_k - \bd{x}^\ast\|_2^2.\] Therefore, \(\{\bd{x}_k\}\) is Fejér monotone with respect to \(\mathcal{X}^\ast\). In particular, the sequence \(\{\bd{x}_k\}\) is bounded. ◻

Let \(\mathcal{C} \subset \bb{R}^n\) be nonempty and closed, and let \(\{\bd{x}_k\} \subset \bb{R}^n\) be Fejér monotone with respect to \(\mathcal{C}\), i.e., \[\|\bd{x}_{k+1} - \bd{z}\|_2 \le \|\bd{x}_k - \bd{z}\|_2 \quad \text{for all } \bd{z} \in \mathcal{C},\ \forall k \ge 0.\] If there exists a subsequence \(\{\bd{x}_{k_j}\}\) and a point \(\bd{x}^\ast \in \mathcal{C}\) such that \[\bd{x}_{k_j} \to \bd{x}^\ast,\] then the whole sequence converges: \[\bd{x}_k \to \bd{x}^\ast.\]

Proof. Fix \(\bd{z} = \bd{x}^\ast \in \mathcal{C}\). By Fejér monotonicity, the sequence \[d_k := \|\bd{x}_k - \bd{x}^\ast\|_2\] is nonincreasing and bounded below by \(0\). Hence, there exists \(\ell \ge 0\) such that \[d_k \to \ell.\]

On the other hand, along the convergent subsequence, \[d_{k_j} = \|\bd{x}_{k_j} - \bd{x}^\ast\|_2 \to 0.\] Therefore, \(\ell = 0\). It follows that \[\|\bd{x}_k - \bd{x}^\ast\|_2 \to 0,\] and hence \(\bd{x}_k \to \bd{x}^\ast\). ◻

Let \(f \in C^1_\beta(\bb{R}^n)\) be convex and let \(\mathcal{X}^\ast := \arg\min f \neq \varnothing\). Assume \(\gamma \in (0,1/\beta]\) and consider the gradient descent iteration \[\bd{x}_{k+1} = \bd{x}_k - \gamma \nabla f(\bd{x}_k).\] Then the sequence \(\{\bd{x}_k\}\) converges to a point \(\bd{x}^\ast \in \mathcal{X}^\ast\).

Proof. By Fejér monotonicity with respect to \(\mathcal{X}^\ast\), the sequence \(\{\bd{x}_k\}\) is bounded. Hence, by the Bolzano–Weierstrass theorem, there exists a subsequence \(\{\bd{x}_{k_j}\}\) and a point \(\bd{x}^\ast \in \bb{R}^n\) such that \[\bd{x}_{k_j} \to \bd{x}^\ast.\]

By the descent lemma, \[f(\bd{x}_{k+1}) \le f(\bd{x}_k) - \frac{\gamma}{2}\|\nabla f(\bd{x}_k)\|_2^2,\] which implies \[\|\nabla f(\bd{x}_k)\|_2 \to 0.\] By continuity of \(\nabla f\), \[\nabla f(\bd{x}^\ast) = 0.\]

Since \(f\) is convex, this implies \(\bd{x}^\ast \in \mathcal{X}^\ast\).

By Fejér monotonicity and Lemma [lem:convergence_fejer_monotone_limit], the sequence \(\{\bd{x}_k\}\) converges to \(\bd{x}^\ast \in \mathcal{X}^\ast\). ◻

Let \(f : \bb{R}^n \to \bb{R}\) be differentiable. The following statements are equivalent:

  1. \(f\) is convex.

  2. For all \(\bd{x}, \bd{y} \in \bb{R}^n\), \[f(\bd{y}) \ge f(\bd{x}) + \langle \nabla f(\bd{x}), \bd{y} - \bd{x} \rangle.\]

  3. For all \(\bd{x}, \bd{y} \in \bb{R}^n\), \[\langle \nabla f(\bd{x}) - \nabla f(\bd{y}), \bd{x} - \bd{y} \rangle \ge 0.\]

Projected Gradient Descent

Let \(\bb{X} \subset \bb{R}^n\) be nonempty, closed, and convex. The projection operator \(\Pi_{\bb{X}} : \bb{R}^n \to \bb{X}\) is defined by \[\Pi_{\bb{X}}(\bd{z}) := \arg\min_{\bd{x} \in \bb{X}} \|\bd{x} - \bd{z}\|_2.\]

Let \(\bb{X} \subset \bb{R}^n\) be closed and convex. Then:

  1. \(\Pi_{\bb{X}}\) is well-defined and single-valued.

  2. \(\Pi_{\bb{X}}\) is nonexpansive: \[\|\Pi_{\bb{X}}(\bd{x}) - \Pi_{\bb{X}}(\bd{y})\|_2 \le \|\bd{x} - \bd{y}\|_2.\]

  3. (Projection inequality) \[\langle \bd{z} - \Pi_{\bb{X}}(\bd{z}), \bd{x} - \Pi_{\bb{X}}(\bd{z}) \rangle \le 0, \quad \forall \bd{x} \in \bb{X}.\]

Proof of (1): Existence. Let \(\mathcal{C} \subset \bb{R}^n\) be nonempty, closed, and convex, and fix \(\bd{x} \in \bb{R}^n\). Choose any point \(\bd{x}_0 \in \mathcal{C}\) and consider the problem: \[\min_{\bd{z} \in \mathcal{C}} \frac12 \|\bd{z} - \bd{x}\|_2^2.\]

Set \[R := \|\bd{x} - \bd{x}_0\|_2 \quad \text{and} \quad \mathcal{K} := \mathcal{C} \cap \overline{B}(\bd{x}, R),\] where \(\overline{B}(\bd{x}, R)\) denotes the closed ball centered at \(\bd{x}\) with radius \(R\).

For any \(\bd{z} \in \mathcal{C}\) such that \(\bd{z} \notin \mathcal{K}\), we must have \(\|\bd{z} - \bd{x}\|_2 > R\). Substituting \(R\), this implies: \[\frac12 \|\bd{z} - \bd{x}\|_2^2 > \frac12 \|\bd{x}_0 - \bd{x}\|_2^2.\] Since \(\bd{x}_0 \in \mathcal{K}\) achieves a strictly lower objective value, no point outside \(\mathcal{K}\) can be a minimizer. Therefore, \[\arg\min_{\bd{z} \in \mathcal{C}} \frac12 \|\bd{z} - \bd{x}\|_2^2 = \arg\min_{\bd{z} \in \mathcal{K}} \frac12 \|\bd{z} - \bd{x}\|_2^2.\]

The set \(\mathcal{K}\) is the intersection of a closed set \(\mathcal{C}\) and a closed, bounded ball, so \(\mathcal{K}\) is closed and bounded (hence compact). Since the function \[\bd{z} \mapsto \frac12 \|\bd{z} - \bd{x}\|_2^2\] is continuous, it attains its minimum on the compact set \(\mathcal{K}\) by the Weierstrass extreme value theorem. Thus, a minimizer exists.

Let \(C \subset \bb{R}^n\) be nonempty, closed, and convex, and let \(f\) be differentiable on an open set containing \(C\). Assume that the problem \[\min_{\bd{x} \in C} f(\bd{x})\] is solvable.

Let \(\{\bd{x}_k\}\) be the sequence generated by projected gradient descent. Then \[f(\bd{x}_k) - \min_{\bd{x} \in C} f(\bd{x}) \le \mathcal{O}\!\left(\frac{1}{k}\right),\] and \[\bd{x}_k \to \bd{x}^\ast, \qquad \bd{x}^\ast \in \arg\min_{\bd{x} \in C} f(\bd{x}).\]

Let \(C \subset \bb{R}^n\) be nonempty, closed, and convex, and let \(f\) be differentiable on an open set containing \(C\). For a point \(\bd{x}^\ast \in C\), the following statements are equivalent:

  1. \[\bd{x}^\ast \in \arg\min_{\bd{x} \in C} f(\bd{x})\]

  2. \[\bd{x}^\ast = \Pi_C\bigl(\bd{x}^\ast - \gamma \nabla f(\bd{x}^\ast)\bigr), \quad \forall \gamma > 0.\]

  3. \[\langle \bd{x} - \bd{x}^\ast, \nabla f(\bd{x}^\ast) \rangle \ge 0, \quad \forall \bd{x} \in C.\]

Let \(C \subset \bb{R}^n\) be nonempty, closed, and convex. For \(\bd{x}_0 \in \bb{R}^n\) and \(\bd{p} \in C\), the following statements are equivalent:

  1. \[\bd{p} = \Pi_C(\bd{x}_0)\]

  2. \[\langle \bd{x}_0 - \bd{p}, \bd{x} - \bd{p} \rangle \le 0, \quad \forall \bd{x} \in C.\]

The inequality states that the angle between the vectors \(\bd{x}_0 - \bd{p}\) and \(\bd{x} - \bd{p}\) is obtuse or right for every \(\bd{x} \in C\). Geometrically, this means that the segment joining \(\bd{x}_0\) to its projection \(\bd{p}\) forms a supporting hyperplane to the set \(C\) at \(\bd{p}\).
