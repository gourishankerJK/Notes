2.1 Examples of variational problems

We begin this chapter by describing a few examples of classical variational problems, i.e., problems in which a path is to be chosen from a given family of admissible paths so as to minimize the value of some functional. These examples will help us motivate the general problem formulation and will serve as benchmarks for the theory that we will be developing.
